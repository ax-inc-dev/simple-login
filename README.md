# シンプルログインページ

HTMLファイル1つで完結するログインページです。

## 特徴

- **単一HTMLファイル**: `index.html` のみで動作
- **外部依存なし**: CSS/JavaScriptすべて内包
- **モダンなデザイン**: グラデーション、アニメーション、レスポンシブ対応

ログインに成功すると、秘密の合言葉「**1000万時間開放**」が表示されます。

## 使い方

### 方法1: ブラウザで直接開く

```bash
open index.html
```

または、`index.html` をブラウザにドラッグ&ドロップ

### 方法2: ローカルサーバーで開く

Python 3の場合:
```bash
cd simple-login
python3 -m http.server 8000
```

Node.jsの場合:
```bash
cd simple-login
npx serve
```

その後、ブラウザで表示されたURLにアクセス
